#PokeManager

###What is it?
---
PokeManager goes through your Pokemon Go account and transfers away all duplicate pokemon with a lower CP!

###Installation
---
To run:

Install deps with

```
npm install
```

then copy config.json.example to config.json and edit settings.

Then run with

```
node index.js
```
